Movie Sentiments Prediction

October 2020
EN.500.112 Gateway Computing: Java

Objectives:

  • Read a .txt file with thousands of movie reviews and their associated ratings (0-4)
  • Tally the number of appearances of each word and the ratings
  • Calculate the predicted rating based on several keywords
  • Rate any input word and determine the positive or negative tone

    • Skills Applied:

  • File I/O, data wrangling, debugging

    • Comments:

    This was an enjoyable project that showed how efficient it is to streamline repetitive data processing with code. I was able to format the input and output data however I want and create a robust tool to handle various tasks. Later in the semester, I started working on data processing at my research lab, which includes lots of file I/O and data clean-up. What I learned from this project turned out to be extremely useful.
