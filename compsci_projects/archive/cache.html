Cache Simulator (x86 Assembly)

March 2022
EN.601.229 Computer Systems Fundamentals

Objectives:

  • Implement a cache with various block sizes and caching techniques (LRU, FIFO)
  • Calculate the number of loads, stores, and total cycles for each scenario
  • Find the best caching technique

    • Skills Applied:

  • Debugging incrementally
  • C++ classes and vectors, preventing memory leaks

    • Comments:

    Write allocate + write through + LRU has the best hit rates, while write allocate + write back has the lowest total cycles. Thus, for a fully-associated cache, there is a tradeoff between slightly improved accuracy and speed.
